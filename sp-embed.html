About This Site

This SharePoint site hosts the outputs of the NSCC AI Curriculum Pipeline — an automated system that uses a locally-hosted large language model (Qwen2.5-32B) running on the NSCC Truro Campus AI Cluster to analyse curriculum documents against accreditation standards.

Each program received two reports: a Standards Compliance Report measuring alignment with NSCC's institutional program outcomes, and a Learning Outcome Alignment Report cross-mapping course-level learning outcomes to program-level outcomes.

The full pipeline processed 15,459 learning outcomes across 148 programs in approximately 27.75 hours of GPU compute — at zero cloud API cost compared to an estimated ~$198 USD using commercial AI services.
